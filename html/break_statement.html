C++ break Statement

In this tutorial, we will learn about the break statement and its working in loops.

break Statement

In C++, the break statement terminates the loop when it is encountered.

The syntax of the break statement is:


          break;

Before we learn about the break statement, make sure you are already familiar about:

  • C++ if...else
  • C++ while and do...while loops
  • C++ for loop

Note: A break statement may only be used within a loop or switch.

Working of break Statement

In the body of a loop statement, the break statement will terminate the loop and "break" out of current parent loop statement.


Example 1: break with for loop


          #include <iostream>
          using namespace std;

          int main() 
          {
            for (int i = 1; i <= 5; i++) 
            {
              if (i == 3)    // break condition
                break;

              cout << i << endl;
            }
            cout << "This line is always printed" << endl;
            return 0;
          }
Output 

          1
          2
          This line is always printed

In the above program, the for loop is used to print the value of i in each iteration, from 1 to 5. Here notice the code:


          if (i == 3)
            break;

This means, when i is equal 3, the break statement immediately terminates the loop. Hence, the output doesn't include values greater than or equal to 3.

Note: The statement break is usually used with decision-making statements.


Example 2: break with while loop


          // program to find the sum of positive numbers
          // if the user enters a negative numbers, break statement ends the loop
          // the negative number entered is not added to sum

          #include <iostream>
          using namespace std;

          int main() 
          {
            int number;
            int sum = 0;

            while (true)
            {
              // take input from the user
              cout << "Enter a number: ";
              cin >> number;

              // break condition
              if (number < 0)
              {
                cout << "Read negative number" << endl;
                break;
                cout << "Breaking out of while loop" << endl;
              }

              // add all positive numbers
              sum += number;
            }

            // display the sum
            cout << "\nThe sum is "<< sum << endl;

            return 0;
          }
Output 

          Enter a number: 1
          Enter a number: 2
          Enter a number: 3
          Enter a number: -5
          Read negative number

          The sum is 6

In the above program, at the beginning of each iteration in the while loop, the user enters a number. As long as the user input a positive number, all the positive number will be summed up. Here, notice that:


          if (number < 0)
          {
            cout << "Read negative number" << endl;
            break;
            cout << "Breaking out of while loop" << endl;
          }

This means, when the user enters a negative number, this if statement is true. Hence, it prints "Read negative number" and break out of the loop. Also note that, since it already jumps out of the loop once encountered the break statement, the statement that prints "Breaking out of while loop" is thus skipped and not executed.

break with Nested Loop

Note that when break is used with nested loops, the break only terminates its "parent" loop.


Example 3: break with nested loop


          #include <iostream>
          using namespace std;

          int main()
          {
            for (int i = 1; i <= 3; i++)
            {
              for (int j = 1; j <= 3; j++)
              {
                if (i == 2) 
                 break;
                 
                cout << "i = " << i << ", j = " << j << endl;
              }
            }
            return 0;
          }
Output 

          i = 1, j = 1
          i = 1, j = 2
          i = 1, j = 3
          i = 3, j = 1
          i = 3, j = 2
          i = 3, j = 3

In the above program, the break statement is executed when i == 2. It terminates the inner loop, and the control flow of the program moves to the outer loop. Hence, the value of i = 2 is never displayed in the output.
