C++ for Loop

In this tutorial, we will learn about the C++ for loop.

In the previous tutorial, we have learnt about while and do...while loops.

for Loop

The syntax of for loop is:


          for (initialization; condition; update) {
            // body of for loop
          }

Here,

  • initialization - initializes variables (iterables) and is executed only once
  • condition - if true, the body of for loop is executed; if false, then the loop is terminated
  • update - updates the value of initialized variables (iterables) and again checks the condition

Note: The iterable variable must only be integer types that are "iterable", i.e. int, short, etc.


Example 1: Display text 5 times


          #include <iostream>
          using namespace std;

          int main() {
            for (int i = 0; i < 5; i++) {
              cout << "Hello World" << endl;
            }

            return 0;
          }
Output 

          Hello World
          Hello World
          Hello World
          Hello World
          Hello World

Here is how the program works

Iteration Variable i < 5 Action
1 i = 0 true Hello World is printed and i is increased to 1
2 i = 1 true Hello World is printed and i is increased to 2
3 i = 2 true Hello World is printed and i is increased to 3
4 i = 3 true Hello World is printed and i is increased to 4
5 i = 4 true Hello World is printed and i is increased to 5
6 i = 5 false Loop terminated

Example 2: Display Numbers from 1 to 5


          #include <iostream>
          using namespace std;

          int main() {

            for (int i = 1; i <= 5; i++) {
              cout << i << " ";
            }

            cout << endl;
            return 0;
          }
Output 

          1 2 3 4 5

Here is how the program works

Iteration Variable i <= 5 Action
1 i = 1 true 1 is printed and i is increased to 2
2 i = 2 true 2 is printed and i is increased to 3
3 i = 3 true 3 is printed and i is increased to 4
4 i = 4 true 4 is printed and i is increased to 5
5 i = 5 true 5 is printed and i is increased to 6
6 i = 6 false Loop terminated

Note: Notice that in Example 2, the iterable i exists only within for loop. It is not accessible from outside the loop. In order to have an iterable accessible outside of the for loop scope, we can write our code similar to the following: 


            int j;
            for (j = 5; j < 10; j++) {
              cout << j << " ";
            }

            // print final j value
            cout << "\nj = " << j << endl;  // prints: j = 10
Output 

            5 6 7 8 9 
            j = 10

Ranged Based for Loop

In C++11, a new range-based for loop was introduced to work with collections such as arrays and vectors. Its syntax is:


          for (variable : collection) {
            // body of loop
          }

Here, for every value in the collection, it is assigned to the variable and the body of the loop is executed.

Example 3: Ranged Based for Loop


          #include <iostream>
          using namespace std;
  
          int main()
          {
            int num_array[] = {1, 2, 3, 4, 5}
  
            for (int n : num_array) {
              cout << n << " ";
            }
            return 0;
          }
Output 

          1 2 3 4 5

In the program above, we have declared and initialized and int array named num_array which has 5 elements.

Then we use a range-based for loop to iterate through all the items in the array.

Infinite for Loop

Similar to while and do...while loop, a for also requires stopping condition. If the condition in a for loop is always true, it runs forever (until memory is full). For example,


          for (int i = 1; i > 0; i++) {
            // code block
          }

Here, the condition is always true which will then run the code for infinite times.


Sometimes, for some reasons programmers would want to implement an infinite for loop, example below is another way of expressing an infinite for loop:


          for (;;) {
            // code block
          }

Here, the for loop is infinite because there is no stopping condition.

Nested for Loop

Nested for loop is a for loop within another for loop.

Example 4: Nested for Loop


          #include <iostream>
          using namespace std;

          int main()
          {
            for (int i = 1; i <= 3; i++)
            {
              for (int j = 1; j <= 3; j++)
              {  
                cout << "i = " << i << ", j = " << j << endl;
              }
              cout << "\n";
            }
            return 0;
          }
Output

          i = 1, j = 2

          i = 2, j = 3
