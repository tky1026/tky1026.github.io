C++ continue Statement

In this tutorial, we will learn about the continue statement and its working with loops.

continue Statement

The continue statement is used to skip the current iteration of the loop and the control of the program goes to the next iteration.

The syntax of continue statement is:


          continue;

Before we learn about the break statement, make sure you are already familiar about:

  • C++ if...else
  • C++ while and do...while loops
  • C++ for loop

Note: A continue statement may only be used within a loop.

Working of continue Statement

In the body of a loop statement, the continue statement will skip the current iteration of the loop and "jump" to the next iteration.


Example 1: continue with for loop

In a for loop, continue skips the current iteration and the control flow jumps to the update expression.


          #include <iostream>
          using namespace std;

          int main() 
          {
            for (int i = 1; i <= 5; i++) 
            {
              if (i == 3)    // continue condition
                continue;

              cout << i << endl;
            }
            cout << "This line is always printed" << endl;
            return 0;
          }
Output 

          1
          2
          4
          5
          This line is always printed

In the example above, we have used the the for loop to print the value of i in each iteration. Here, notice the code:


          if (i == 3)
            continue;

This means when i is equal to 3, the continue statement skips the current iteration and start the next iteration. Therefore, the statement that prints out i is skipped.

Then, in the update expression, i is incremented to 4 and the condition is evaluated again. Hence 4 and 5 are printed in the next two iteration.

Note: The statement continue is usually used with decision-making statements.


Example 2: continue with while loop

In a while loop, continue skips the current iteration and control flow of the program jumps back to the while condition.


          // program to calculate positive numbers till 50 only
          // if the user enters a negative number,
          // that number is skipped from the calculation
          // negative number -> loop terminate
          // numbers above 50 -> skip iteration
  
          #include <iostream>
          using namespace std;
  
          int main()
          {
            int number = 0;
            int sum = 0;
  
            while (number >= 0)
            {
              // add all positive numbers
              sum += number;
  
              // take input from the user
              cout << "Enter a number: ";
              cin >> number;
  
              // continue condition
              if (number > 50)
              { 
                cout << "The number is greater than 50 and won't be calculated."<< endl;
                number = 0;  // reset the number to 0 s.t. it will not be added
                continue;
              }
  
              cout << "---" << endl;
            }
  
            // display the sum
            cout << "\nThe sum is "<< sum << endl;
            return 0;
          }
Output 

          Enter a number: 5
          ---
          Enter a number: 0
          ---
          Enter a number: 56
          The number is greater than 50 and won't be calculated.
          Enter a number: 7
          ---
          Enter a number: -1
          ---
  
          The sum is 12

In the above program, the loop will keep adding the positive number smaller than 50 input by the user. Note the use of the continue statement:


          if (number > 50)
            continue;

When the user enters a number greater than 50, the continue statement skips the current iteration and whatever is left behind. Then the control flow of the program goes to the contidion of the while loop. Notice that in the iteration where the continue is executed, the --- is not printed.

continue with Nested Loop

Note that when continue is used with nested loops, the continue only skip the iteration of its "parent" loop.


Example 3: continue with nested loop


            #include <iostream>
            using namespace std;
  
            int main()
            {
              for (int i = 1; i <= 3; i++)
              {
                for (int j = 1; j <= 3; j++)
                {
                  if (j == 2) 
                   continue;
                   
                  cout << "i = " << i << ", j = " << j << endl;
                }
              }
              return 0;
            }
Output 

            i = 1, j = 1
            i = 1, j = 3
            i = 2, j = 1
            i = 2, j = 3
            i = 3, j = 1
            i = 3, j = 3

Note that when j is equal to 2, the iteration of the inner loop is skipped. Hence, the value of j = 2 is never displayed in the output.
