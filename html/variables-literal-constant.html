C++ Variables, Literals and Constants

In this tutorial, we will learn about variables, literals, and constants in C++.

C++ Variables

In programming, a variable is a container (storage area) to hold data.

To indicate the storage area, each variable should be given a unique name (identifier).


          int age = 21;

Here, age is a variable of the int data type, and we have assigned an integer value 14 to it.

Note: The int data type suggests that the variable can only hold integers. Similarly, we can use the float and double data type if we have to store decimals and exponentials.

Rules for naming a variable

  • A variable name can only have alphabets, numbers, and the underscore _.
  • A variable name cannot begin with a number.
  • Variable names should not begin with an uppercase character.
  • Recommended style for a variable name is camel_case
  • A variable name cannot be a keyword. For example, int is a keyword that is used to denote integers.
  • A variable name can start with an underscore. However, it's not considered a good practice.

Note: We should try to give meaningful names to variables. For example, first_name is a better variable name than fn.

C++ Literals

Literals are data used for representing fixed values, they are fixed values that the program may not alter. They can be used directly in the code. For example: 1, 2.5, 'c' etc.

Here, 1, 2.5 and 'c' are literals. Why? You cannot assign different values to these terms.

Here's a list of different literals in C++ programming.

1. Integer Literals

An integer is a numeric literal(associated with numbers) without any fractional or exponential part. There are three types of integer literals in C programming:

  • decimal (base 10)
  • octal (base 8)
  • hexadecimal (base 16)

For example:


          Decimal: 0, -9, 22 etc
          Octal: 021, 077, 033 etc
          Hexadecimal: 0x7f, 0x2a, 0x521 etc 
          Illegal: 078  // 8 is not an octal digit

In C++ programming, octal starts with a 0, and hexadecimal starts with a 0x.

2. Floating-point Literals

A floating-point literal has an integer part, a decimal point, a fractional part, and an exponent part. You can represent floating point literals either in decimal form or exponential form.

While representing using decimal form, you must include the decimal point, the exponent, or both and while representing using exponential form, you must include the integer part, the fractional part, or both. The signed exponent is introduced by e or E.

For example:


          -3.14159
          0.0000234
          0.234e-5

3. Boolean Literals

There are two Boolean literals and they are part of standard C++ keywords.

  • true representing true
  • false representing false

4. Characters Literals

A character literal is created by enclosing a single character inside single quotation marks. For example: 'a', 'm', 'F', '2', '}' etc.

A character literal can be a plain character (e.g., 'x'), an escape sequence (e.g., '\t'), or a universal character (e.g., '\u02C0').

5. Escape Sequence

Sometimes, it is necessary to use characters that cannot be typed or has special meaning in C++ programming. For example, newline (enter), tab, question mark, etc.

In order to use these characters, escape sequences are used.

Escape Sequences Characters
\b Backspace
\f Form feed
\n Newline
\r Return
\t Horizontal tab
\v Vertical tab
\\ Backslash
\' Single quotation mark
\" Double quotation mark
\? Question mark
\0 Null Character

6. String Literals

A string literal is a sequence of characters enclosed in double-quote marks.

For example:

"good" string constant
"" null string constant
" " string constant of a white space
"x" string constant having a single character
"Earth is round\n" prints string with a newline

C++ Constants

In C++, we can create variables whose value cannot be changed. For that, we use the const keyword. Here's an example:


          const int LIGHT_SPEED = 299792458;
          LIGHT_SPEED = 2500 // Error! LIGHT_SPEED is a constant.

Here, we have used the keyword const to declare a constant named LIGHT_SPEED. If we try to change the value of LIGHT_SPEED, we will get an error.

There are two ways to define constants:

  • Using #define preprocessor
  • Using const keyword

          #define DAYS_IN_WEEK 7
          const float PI = 3.1415

A constant can also be created using the #define preprocessor directive. We will learn about it in detail in the C++ Macros tutorial.
