C++ Function Template

In this tutorial, we will learn about class templates in C++.

Templates are powerful features of C++ which allows us to write generic programs.

We can create a single function to work with different data types by using a template.

Function Template

A function template starts with the keyword template followed by template parameter(s) inside <> which is followed by the function definition.


          template <typename T>
          T functionName(T parameter1, T parameter2, ...) {
            // code
          }

Here, T is a template argument or generic data type that accepts different data types (int, float, etc.), and typename is a keyword.

When an argument of a data type is passed to functionName(), the compiler generates a new version of functionName() for the given data type. This process is being done behind the scene during compilation.

Calling Function Template

Once we've declared and defined a function template, we can call it in other functions or templates (such as the main() function) with the following syntax:


          functionName<dataType>(parameter1, parameter2, ...);

Example


          #include <iostream>
          using namespace std;
  
          template <typename T>
          T add(T num1, T num2) {
            return (num1 + num2);
          }

          int main() {

            // calling with int parameters
            int result1 = add<int>(2, 3);
            cout << result1 << endl;

            // calling with double parameters
            double result2 = add<double>(2.2, 3.3);
            cout << result2 << endl;

            return 0;
          }
Output 

          5
          5.5

When the compiler encounters this call to a template function, it uses the template to automatically generate a function replacing each appearance of generic type T by the type passed as the actual template parameter (int and double in this case) and then calls it. This process is automatically performed by the compiler and is invisible to the programmer.
