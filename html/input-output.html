C++ Basic Input/Output

In this tutorial, we will learn to use the cin object to take input from the user, and the cout object to display output to the user.

Note: Both cin and cout object are defined in C++ standard library iostream.

C++ Output

In C++, cout sends formatted output to standard output devices, such as the screen. We use the cout object along with the << operator for displaying output.

Example 1: String Output


          #include <iostream>
          using namespace std;
          
          int main() {
              // prints the string enclosed in double quotes
              cout << "This is C++ Programming";
              return 0;
          }

Output


          This is C++ Programming
  • We first include the iostream header file that allows us to display output.
  • The cout object is defined inside the std namespace. To use the std namespace, we used the using namespace std; statement.
  • Every C++ program starts with the main() function. The code execution begins from the start of the main() function.
  • cout is an object that prints the string inside quotation marks " ". It is followed by the << operator.
  • return 0; is the "exit status" of the main() function. The program ends with this statement, however, this statement is not mandatory.

Note: If we don't include the using namespace std; statement, we need to use std::cout instead of cout.

Namespaces in C++ is a declarative region that provides a scope to the identifiers (types, variables, classes, objects, functions, etc.). It is used to divide the global scope into the sub-scopes.


            std::cout << "This is C++ Programming";

Example 2: Numbers, Characters and Boolean Output

Numbers, characters, and boolean can also be directly printed using cout.

  • Numbers, integers and floating-points, can be printed without using a quotation marks.
  • Characters need to be enclosed with a single quotation mark ' '.
  • Booleans are printed as 1 for true and 0 for false.

          cout << 123.45 << endl;
          cout << 'C' << endl;
          cout << true << " " << false << endl;

Output


          123.45
          C
          1 0

Note: The endl manipulator is used to insert a new line. That's why each output above is displayed in a new line.

Example 3: Variables Output

cout can also prints out data stored in a variable.


          bool var_bool = false;
          int var_int = 12;
          float var_float = 3.4;
          char var_char = 'A';
          string var_str = "Hello";

          cout << "bool  : " << var_bool << endl;
          cout << "int   : " << var_int << endl;
          cout << "float : " << var_float << endl;
          cout << "char  : " << var_char << endl;
          cout << "string: " << var_str << endl;

Output


          bool  : 0
          int   : 12
          float : 3.4
          char  : A
          string: Hello

Note: The << operator applied to an output stream is known as insertion operator. It can be used to print different variables, strings, etc. into a single formatted statement.

C++ Input

In C++, cin takes formatted input from standard input devices such as the keyboard. We use the cin object along with the >> operator for taking input.

Note: The >> operator applied to an input stream is known as extraction operator. It is used to read from the formatted input and store the extracted data into a variable.

Example 4: Integer Input


          #include <iostream>
          using namespace std;
          
          int main() {
              int num;
              cout << "Enter an integer: ";
              cin >> num;   // Taking input
              cout << "The number is: " << num;
              return 0;
          }

Output


          Enter an integer: 70
          The number is: 70

Here, we used


          cin >> num;

to take input from the user, and the input is stored in the variable num.

Note: If we don't include the using namespace std; statement, we need to use std::cin instead of cin.

Example 5: Taking Multiple Inputs


          #include <iostream>
          using namespace std;
          
          int main() {
              char a;
              int num;
          
              cout << "Enter a character and an integer: ";
              cin >> a >> num;
          
              cout << "Character: " << a << endl;
              cout << "Number: " << num << endl;
          
              return 0;
          }

Output


          Enter a character and an integer: J 18
          Character: J
          Number: 18
