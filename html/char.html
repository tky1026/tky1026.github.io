C++ char

In this tutorial, we will learn about the char data type in C++.

In C++, char keyword is used to declare character type variables. A character variable can store only a single character.

Example 1: Printing a char variable


          #include <iostream>
          using namespace std;
          int main() {
              // initializing a variable
              char ch = 'h';
          
              // printing the  variable
              cout << "Character = " << ch << endl;
          
              return 0;
          }

Output


          Charcter = h

In the example above, we have declared a character type variable named ch, and assigned the character h to it.

Note: In C and C++, a character should be inside single quotation marks, while double quotation marks for a string.

ASCII Value

In C and C++, an integer (ASCII value) is stored in char variables rather than the character itself. For example, if we assign 'h' to a char variable, 104 is stored in the variable rather than the character itself. It's because the ASCII value of 'h' is 104.

The table below showing the ASCII values of characters of A, Z, a, z and 5.

Characters ASCII Values
A 65
Z 90
a 97
z 122
5 53

To learn more about ASCII value, visit the ASCII Chart.

Example 2: Printing ASCII value of a character


          #include <iostream>
          using namespace std;
          
          int main() {
              char ch = 'h';
          
              // Printing the corresponding ASCII of a character 
              // Notice the use of int() to get an integer, this is called type casting
              cout << "ASCII value = " << int(ch) << endl;
          
              return 0;
          }

Output


          ASCII value = 104

Here, get the corresponding ASCII value of a character by using int() when we print it.

For a char data type variable, we can either assign an ASCII value (from 0 to 127), or a single quoted character itself to it.

Example 3: Assigning a character


          // assigning the character itself
          char ch1 = 'A';
          std::cout << "ch1 Character   = " << ch1 << std::endl;
          std::cout << "ch1 ASCII value = " << int(ch1) << std::endl;

          // assigning the ascii value
          char ch2 = 65;
          std::cout << "ch2 Character   = " << ch2 << std::endl;
          std::cout << "ch2 ASCII value = " << int(ch2) << std::endl;

Output


          ch1 Character   = A
          ch1 ASCII value = 65
          ch2 Character   = A
          ch2 ASCII value = 65

We can see that both assigning the character itself to ch1 and assigining the integer ASCII value to ch2 give us the same result.

Note: If we assign '8' (quotation marks) to a char variable, we are storing 56 (its ASCII value). However, if we assign 8 (without quotation marks) to a char variable, we are storing the ASCII value 8 (a backspace character).

Escape Sequences

Some characters have special meaning in C++, such as single quote ', double quote ", backslash \ and so on. We cannot use these characters directly in our program.

For example: 


          // This code shows an error
          char character = ''';

Here, we are trying to store a single quote character ' in a variable. But this code shows a compilation error.

So how can we use those special characters?

To solve this issue, C++ provides special codes known as escape sequences with a backslash key \. Now with the help of escape sequences, we can write those special characters as they are.

For example: 


          char character = ' \' ';

Here, \' is an escape sequence that allows us to print or store a single quote in the variable.

The table below lists escape sequences of C++.

Escape Sequences Character
\b Backspace
\f Form feed
\n Newline
\r Return
\t Horizontal tab
\v Vertical tab
\\ Backslash
\' Single quotation mark
\"e> Double quotation mark
\? Question mark
\0 Null character

Example 4: Using escape sequences


          char character1 = 'A';
          char character2 = '\t';
          char character3 = '5';
          char character4 = '\n';
          char character5 = 97;

          cout << character1;   // A
          cout << character2;   // horizontal tab
          cout << character3;   // 5
          cout << character4;   // new line
          cout << character5;   // a

Output


          A       5
          a
