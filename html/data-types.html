C++ Data Types

In this tutorial, we will learn about basic data types such as int, float, char, etc.

In C++, data types are declarations for variables. This determines the type and size of data associated with variables. For example:


          int age = 18
          bool is_adult = true;

Here, age is a variable of type int while is_adult is a variable of type bool.

While writing program in any language, you need to use various variables to store various information. Variables are nothing but reserved memory locations to store values. This means that when you create a variable you reserve some space in memory.

You may like to store information of various data types like character, wide character, integer, floating point, double floating point, boolean etc. Based on the data type of a variable, the operating system allocates memory and decides what can be stored in the reserved memory.

C++ Fundamental Data Types

In C++ there are seven basic data types / primitive data types.

Data Type Meaning Size(Bytes)
int Integer 4
float Floating-point 4
double Double Floating-point 8
char Character 1
wchar_t Wide Character 2
bool Boolean 1
void Empty / Valueless 0

1. int

  • The int keyword is used to indicate integers.
  • Its size is 4 bytes and it can store values from -2147483648 to 2147483647.

            int salary = 85000;

2. float and double

  • float and double are used to store floating-point numbers (decimals and exponentials).
  • The size of float is 4 bytes and the size of double is 8 bytes. Hence, double has two times the precision of float.

            float area = 12.34;
            double volume = 56.789;
            double two_million = 2e6 // 2e6 or 2E6 is equal to 2*10^6

3. char

  • Keyword char is used for character.
  • char is 1 byte in size.
  • Characters in C++ are enclosed inside single quotes ' '.

            char test = 'k';

Note: In C++, an integer value is stored in a char variable rather than the character itself, the integer value is called ASCII value. C++ char

4. wchar_t

  • Wide character wchar_t is similar to the char data type, except its size is 2 bytes instead of 1.
  • It is used to represent characters that require more memory to represent them than a single char.
  • Mostly the wchar_t datatype is used when international languages like Japanese are used.

            wchar_t hello[] = L"Привет мир"; // hello world in russian

5. bool

  • The bool data type only has one of the two possible values: true of false.
  • Booleans are used in conditional statements and loops.

            bool cond = false;

6. void

  • The void keyword indicates an absence of data. It means "nothing" or "no value".
  • We cannot declare variables of void type.

C++ Type Modifiers

We can further modify some of the fundamental data types by using type modifiers. There are 4 type modifiers in C++.

  1. signed
  2. unsigned
  3. short
  4. long

We can modify the following data types with the above modifiers:

  • int
  • double
  • char

Derived Data Types

Data types that are derived from fundamental data types are derived types. For example: arrays, pointers, function types, structures, etc.
